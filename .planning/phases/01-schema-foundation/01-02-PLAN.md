SchemaVersionMixin pattern:
```python
import re
from pydantic import BaseModel, field_validator

SEMVER_RE = re.compile(r"^\d+\.\d+\.\d+$")

class SchemaVersionMixin(BaseModel):
    schema_version: str  # Required — no default

    @field_validator("schema_version")
    @classmethod
    def validate_semver(cls, v: str) -> str:
        if not SEMVER_RE.match(v):
            raise ValueError(f"schema_version must be semver (e.g. '1.0.0'), got: {v!r}")
        return v
```

Discriminated union for TypeDef (ScalarType must be last — it's the fallback):
```python
from typing import Annotated, Literal, Union
from pydantic import Field

TypeDef = Annotated[
    Union[EnumType, StructType, ScalarType],
    Field(discriminator="base")
]
```
Note: Use model_validator on ScalarType to check base is one of the five xtUML primitives
(Boolean|Integer|Real|String|UniqueID) since it can't use Literal for those.

Guard all-or-none validator on StateDiagramFile:
```python
from pydantic import model_validator
from itertools import groupby

@model_validator(mode='after')
def check_guard_consistency(self) -> 'StateDiagramFile':
    key = lambda t: (t.from_state, t.event)
    for (from_state, event), group in groupby(
        sorted(self.transitions, key=key), key=key
    ):
        ts = list(group)
        guarded = [t for t in ts if t.guard is not None]
        if guarded and len(guarded) != len(ts):
            raise ValueError(
                f"Transitions from '{from_state}' on '{event}': "
                f"mix of guarded and unguarded — all must have guards"
            )
    return self
```

model_validator for associative formalizes:
```python
@model_validator(mode='after')
def check_associative_has_formalizes(self) -> 'ClassDef':
    if self.stereotype == "associative" and self.formalizes is None:
        raise ValueError(
            f"Class {self.name!r}: stereotype 'associative' requires a 'formalizes' field"
        )
    return self
```

- `description: str`

**BridgeEntry:**
- `from_domain: str` (key is `from` in YAML — use `Field(alias="from")` and `model_config = ConfigDict(populate_by_name=True)`)
- `to: str`
- `operations: list[BridgeOperation]`

**BridgeOperation:**
- `name: str`
- `params: list[OperationParam] = []`
- `return_type: str | None = None` (key is `return` in YAML — use `Field(alias="return")`)

**OperationParam:**
- `name: str`
- `type: str`

**TypesFile(SchemaVersionMixin):**
- `schema_version: str`
- `domain: str`
- `types: list[TypeDef]`

**TypeDef discriminated union** (ScalarType last — fallback):
- `EnumType`: name, `base: Literal["enum"]`, values: list[str], description: str | None
- `StructType`: name, `base: Literal["struct"]`, description: str | None, fields: list[StructField]
- `ScalarType`: name, `base: str`, units: str | None, range: list[float] | None, description: str | None
  — model_validator checks `base` is one of: Boolean, Integer, Real, String, UniqueID

**StructField:**
- `name: str`
- `type: str`

**ClassDiagramFile(SchemaVersionMixin):**
- `schema_version: str`
- `domain: str`
- `classes: list[ClassDef] = []`
- `associations: list[Association] = []`
- `bridges: list[BridgeStanza] = []`

**ClassDef:**
- `name: str`
- `stereotype: Literal["entity", "active", "associative"]`
- `specializes: str | None = None` (R-number of supertype partition this class is a subtype of)
- `formalizes: str | None = None` (R-number for associative stereotype)
- `partitions: list[SubtypePartition] | None = None`
- `attributes: list[Attribute] = []`
- `methods: list[Method] = []`

**Attribute:**
- `name: str`
- `type: str`
- `identifier: bool = False`
- `referential: str | None = None` (R-number if referential)

**Method:**
- `name: str`
- `scope: Literal["instance", "class"]`
- `params: list[MethodParam] = []`
- `return_type: str | None = None` (key is `return` in YAML — alias)
- `action: str | None = None`

**MethodParam:**
- `name: str`
- `type: str`

**SubtypePartition:**
- `name: str` (R-number, e.g. "R2")
- `discriminator: str`
- `subtypes: list[str]`

**Association:**
- `name: str` (R-number)
- `point_1: str`
- `point_2: str`
- `mult_1_to_2: str` (key is `1_mult_2` in YAML — alias; values: "1", "1..M", "0..1", "M")
- `mult_2_to_1: str` (key is `2_mult_1` in YAML — alias)
- `phrase_1_to_2: str` (key is `1_phrase_2` in YAML — alias)
- `phrase_2_to_1: str` (key is `2_phrase_1` in YAML — alias)

**BridgeStanza** — use discriminated union on direction field:
- `RequiredBridge`: `to_domain: str`, `direction: Literal["required"]`, `operations: list[str]`
- `ProvidedBridge`: `to_domain: str`, `direction: Literal["provided"]`, `implementations: list[BridgeImplementation]`

**BridgeImplementation:**
- `name: str`
- `action: str`

**StateDef:**

```python
import pytest
from pydantic import ValidationError
from mdf_server.schema.yaml_schema import (
    DomainsFile, ClassDiagramFile, StateDiagramFile, TypesFile
)

VALID_DOMAINS = {
    "schema_version": "1.0.0",
    "domains": [
        {"name": "Hydraulics", "type": "application", "description": "Controls valves"},
        {"name": "Timer", "type": "realized", "description": "Wraps OS timers"},
    ],
    "bridges": [
        {
            "from": "Hydraulics",
            "to": "Timer",
            "operations": [
                {"name": "start_timer", "params": [{"name": "duration", "type": "Integer"}], "return": "UniqueID"},
                {"name": "cancel_timer", "params": [{"name": "timer_id", "type": "UniqueID"}], "return": None},
            ],
        }
    ],
}

def test_valid_domains_yaml_accepted():
    result = DomainsFile.model_validate(VALID_DOMAINS)
    assert result.schema_version == "1.0.0"
    assert len(result.domains) == 2
    assert len(result.bridges) == 1

def test_missing_schema_version_rejected():
    bad = {k: v for k, v in VALID_DOMAINS.items() if k != "schema_version"}
    with pytest.raises(ValidationError) as exc_info:
        DomainsFile.model_validate(bad)
    assert "schema_version" in str(exc_info.value)

def test_invalid_semver_rejected():
    bad = {**VALID_DOMAINS, "schema_version": "1.0"}  # Missing patch
    with pytest.raises(ValidationError):
        DomainsFile.model_validate(bad)
```
